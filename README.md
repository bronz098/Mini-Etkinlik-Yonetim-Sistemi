# 🎉 Mini Event Management System

Selam 👋
Bu proje, Visual Studio kullanarak geliştirdiğim küçük çaplı bir “Mini Etkinlik Yönetim Sistemi” projesi.

Asıl amacım Windows Forms tarafında kendimi geliştirmek ve temel masaüstü otomasyon mantığını daha iyi öğrenmekti. Bu süreçte DataGridView kullanımı, button eventleri, form kontrolleri gibi birçok konuda pratik yapmış oldum.

Uygulama üzerinden:

* Etkinlik eklenebiliyor
* Etkinlikler listelenebiliyor
* Seçilen etkinlik silinebiliyor
* Form alanları temizlenebiliyor

Küçük ama geliştirilebilir bir başlangıç projesi oldu 🚀

# 🎉 Mini Event Management System

Hey 👋
This is a small-scale “Mini Event Management System” project that I developed using Visual Studio.

My main goal with this project was to improve myself in Windows Forms development and better understand the logic behind desktop automation systems. During the process, I had the chance to practice topics like DataGridView usage, button events, and form controls.

With this application, users can:

* Add events
* List events
* Delete selected events
* Clear form inputs

It’s a small but expandable beginner project 🚀
